How to Upload new Projects using Folder System:
  1. Create a folder named exactly like the Folder Name (slug) inside the projects/ directory on your computer.
  2. Copy the high-res project images into your new folder, naming them exactly as specified in the form above (e.g. 1.jpg, 2.gif).
  3. Fill out the form above, click "Generate" and then click "Download metadata.json". Save this file inside your new folder.
  4. Click "Update projects.json list" to download a fresh root index file, and save it in the root folder of your website.
  5. Commit your files to your GitHub repository. In seconds, GitHub will host the new project!
